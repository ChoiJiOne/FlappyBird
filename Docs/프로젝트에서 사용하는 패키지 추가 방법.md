# 프로젝트에서 사용하는 패키지 추가 방법

이 문서는 `FlappyBird` 게임 개발 프로젝트에서 사용하는 패키지를 추가하는 방법에 대한 문서입니다.

## 프로젝트에서 사용하는 패키지 목록

이 프로젝트에서 사용하는 패키지 목록은 다음과 같습니다.

- [Raylib-cs](https://github.com/raylib-cs/raylib-cs)
- [Microsoft.Diagnostics.NETCore.Client](https://github.com/dotnet/diagnostics)
- [Microsoft.Data.Sqlite](https://www.nuget.org/packages/Microsoft.Data.Sqlite)

## 패키지 버전

각 패키지 별 버전은 다음과 같습니다.

- `Raylib-cs` : `version 7.0.0`
- `Microsoft.Data.Sqlite` : `version 9.0.1`
- `Microsoft.Diagnostics.NETCore.Client` : `version 0.2.553101`

## 패키지 설치

프로젝트에서 사용하는 패키지를 추가하기 위해서는 CLI에서 다음 명령어를 수행합니다.

### Raylib-cs

```PowerShell
dotnet add package Raylib-cs --version 7.0.0
```

### Microsoft.Data.Sqlite

```PowerShell
dotnet add package Microsoft.Data.Sqlite --version 9.0.1
```

### Microsoft.Diagnostics.NETCore.Client

```PowerShell
dotnet add package Microsoft.Diagnostics.NETCore.Client --version 0.2.553101
```
